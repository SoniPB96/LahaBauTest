# Laha V8 – Website-Workspace ohne Rechnerfokus

Dieser Arbeitsstand enthält die restliche Website als separaten Fokus.

## Enthalten
- src/App.jsx
- src/main.jsx
- src/styles.css
- src/config/siteConfig.js

## Zweck
Diesen ZIP-Stand in einem separaten Chat hochladen, wenn an den restlichen Bereichen der Website gearbeitet werden soll.

## Wichtig
- Der Kostenrechner wird in einem anderen Chat separat weiterentwickelt
- Änderungen hier sollen mit dem Rechner kompatibel bleiben
- Keine unnötigen Eingriffe in die Rechnerlogik
